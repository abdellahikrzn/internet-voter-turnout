# internet-voter-turnout
Analysis of internet access vs voter turnout in African countries
Internet Access vs Voter Turnout

Analyzing whether internet access influences voter turnout across African countries.
Built with Python (pandas, seaborn, scipy) using merged socio-economic datasets.
Findings show a weak relationship (low R²), suggesting internet access alone is not a strong predictor of participation.
Includes visualizations (scatter, regression, bar charts) and statistical analysis.
